# 🥗 Nutri-Flow

Nutri-Flow is a modern nutrition and healthy lifestyle web application designed to help users explore healthy food choices, track nutrition-related information, and maintain a balanced lifestyle through a clean and user-friendly interface.
---

## 🌐 Live Demo

🔗 https://nutriflow1.netlify.app/

* 🍎 Clean and responsive UI design
* 📱 Mobile-friendly layout

# 💡 Project Goal

The main goal of Nutri-Flow is to create an easy-to-use and visually appealing nutrition platform that promotes healthy eating habits and provides users with a smooth digital experience.

This project was built as part of learning and improving frontend development skills while working on a real-world themed application.

---

# ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/ozairmh1/Nutri-Flow.git
```

2. Navigate to the project folder

```bash
cd Nutri-Flow
```

3. Open the project

You can directly open the `index.html` file in your browser.
